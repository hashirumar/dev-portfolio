# AI Headshot Generator

A SaaS product that converts casual portraits into professional-quality headshots leveraging AI.

## Features

- Upload photos and generate multiple headshot styles
- Background removal and image enhancement
- Payment integration via Stripe
- Tech stack: React frontend, Node.js backend, MongoDB database
