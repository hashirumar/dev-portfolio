# Twitter Clone

A full-stack social media platform mimicking Twitter's core features, including real-time updates.

## Features

- Real-time feeds and notifications via WebSocket
- Persistent storage using Redis cache and MongoDB
- User profiles, tweets, likes, and follows
- Built with React, Express.js, WebSocket, Redis
